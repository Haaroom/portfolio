PYTHON

DATA ANALYSIS

  • pandas

Pandas is a Python library used for data manipulation and analysis.
It provides data structures like DataFrames and Series,
which are designed to make working with structured data easy and intuitive.
Pandas is built on top of NumPy and is widely used in data science, machine learning,
and other fields that require data processing.

DATA VISUALIZATION

  • matplotlib

Matplotlib is a Python library used for creating static, animated, and interactive visualizations.
It provides a wide range of functionalities for visualizing data in various formats, making it an essential tool for data analysis and exploration.
It is built upon NumPy and is designed to work well with the SciPy stack.
Matplotlib can be used in Python scripts, IPython shells, web application servers, and various GUI toolkits.

DASH BOARD

  • dash

Dash is a Python library that simplifies building web applications, particularly those focused on data visualization and analytics.
It allows developers to create interactive web dashboards and data apps using pure Python, without the need for extensive front-end development knowledge.
Dash leverages libraries like React.js, Flask, and Plotly.js to create a user-friendly and powerful framework.
